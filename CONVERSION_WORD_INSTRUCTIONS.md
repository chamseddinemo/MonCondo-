# Instructions pour convertir en Word

## Option 1: Utiliser Pandoc (Recommandé)

1. Installer Pandoc: https://pandoc.org/installing.html
2. Exécuter la commande:
```powershell
pandoc DOCUMENTATION_COMPLETE_BACKEND_FRONTEND.md -o DOCUMENTATION_COMPLETE_BACKEND_FRONTEND.docx
```

## Option 2: Copier-coller dans Word

1. Ouvrir `DOCUMENTATION_COMPLETE_BACKEND_FRONTEND.md` dans un éditeur de texte
2. Copier tout le contenu (Ctrl+A, Ctrl+C)
3. Ouvrir Microsoft Word
4. Coller le contenu (Ctrl+V)
5. Word convertira automatiquement le Markdown en format Word

## Option 3: Utiliser un convertisseur en ligne

1. Aller sur https://www.markdowntoword.com/
2. Uploader le fichier `DOCUMENTATION_COMPLETE_BACKEND_FRONTEND.md`
3. Télécharger le fichier Word généré

## Option 4: Utiliser VS Code

1. Installer l'extension "Markdown PDF" dans VS Code
2. Ouvrir `DOCUMENTATION_COMPLETE_BACKEND_FRONTEND.md`
3. Clic droit → "Markdown PDF: Export (docx)"
